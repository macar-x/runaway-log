# Getting Started with RunawayLog

Welcome to RunawayLog! This guide will help you quickly get started with tracking your escape dreams. Follow these simple steps to begin your journey.

## 1. Accessing RunawayLog

### Online Access

The easiest way to use RunawayLog is through the online version:

1. Open your web browser
2. Navigate to [RunawayLog](https://runawaylog.example.com)
3. The app will load automatically

### Local Installation

If you prefer to run RunawayLog locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/macar-x/runaway-log.git
   cd runaway-log
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:5173`

### Docker Installation

For containerized deployment:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/macar-x/runaway-log.git
   cd runaway-log
   ```

2. **Start with Docker Compose:**
   ```bash
   docker compose up -d
   ```

3. **Open your browser:**
   Navigate to `http://localhost:8080`

## 2. First-Time Usage

When you first open RunawayLog, you'll see a simple interface with a prominent "Run" button in the center. Here's what to do next:

1. **Click the "Run" button:**
   This records your first escape dream hit. You'll see a smooth animation and a motivational quote.

2. **Explore the interface:**
   - **Top Navigation:** Includes Settings and Logout buttons
   - **Main Area:** Shows the "Run" button, statistics, and calendar
   - **Side Panel:** Displays your hit logs

3. **Check the statistics:**
   After your first hit, the statistics dashboard will update to show your progress.

4. **View the calendar:**
   The calendar heatmap will display your first hit with a colored square.

## 4. Using Core Features

### Recording Hits

1. Simply click the "Run" button whenever you feel the urge to escape
2. Watch the animation and read the motivational quote
3. Your hit will be recorded and displayed in the statistics and calendar

### Viewing Statistics

1. Look at the statistics cards below the Run button
2. Each card shows different metrics:
   - Total hits
   - Average hits per day
   - Current streak
   - Longest streak
   - Most active day
   - Peak time
   - Highest hit day

### Exploring the Calendar

1. Hover over any colored square on the calendar to see exact hit counts
2. Click on a day to see all hits from that day in the logs panel
3. Use the theme selector in settings to change the calendar colors

### Managing Your Logs

1. The side panel shows all your hits in chronological order
2. Each entry includes:
   - Timestamp
   - Motivational quote
   - Hit ID
3. Use the search bar to find specific entries
4. Sort by date or other criteria

## 5. Customizing Your Experience

### Dark Mode

1. Open the settings menu
2. Toggle the dark mode switch
3. The theme will change immediately
4. Your preference will be saved for future visits

### Calendar Theme

1. Open the settings menu
2. Select "Calendar Theme"
3. Choose from the available themes:
   - GitHub Green (default)
   - Ocean Blue
   - Sunset Orange
   - Purple Dream
   - Fire Red
   - Monochrome Gray
4. Your selection will be applied instantly

### Timezone Settings

1. Open the settings menu
2. Select "Timezone"
3. Choose your preferred timezone from the dropdown
4. Your hits will be displayed in the correct timezone

## 6. Installing as a PWA

RunawayLog is a Progressive Web App (PWA), which means you can install it on your device:

### On Desktop

1. Open RunawayLog in Chrome or Edge
2. Click the install icon in the address bar
3. Follow the prompts to install
4. RunawayLog will appear in your app launcher

### On Mobile

1. Open RunawayLog in Safari (iOS) or Chrome (Android)
2. Tap the share button
3. Select "Add to Home Screen"
4. Follow the prompts to install
5. RunawayLog will appear on your home screen

## 7. Exporting and Importing Data

### Exporting Your Data

1. Open the settings menu
2. Click "Export Data"
3. A JSON file will be downloaded to your device
4. Store this file in a safe location for backup

### Importing Data

1. Open the settings menu
2. Click "Import Data"
3. Select the JSON file from your device
4. Your data will be imported and merged with existing data
